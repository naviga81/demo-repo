# outputs/audit-reports/

This directory stores audit reports generated by the Audit Agent.

Each file is named `<work-item-id>_AuditReport.md` and corresponds to one pipeline run.

Audit reports are also posted as comments on the ADO work item by the Supervisor Agent.

**Template:** [`pipeline/templates/audit-report-template.md`](../../pipeline/templates/audit-report-template.md)

> This directory is gitignored in production. Contents are pipeline-generated.
