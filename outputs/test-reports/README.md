# outputs/test-reports/

This directory stores test result reports generated by the Test Agent.

Each file is named `<work-item-id>_TestResults.md` and corresponds to one pipeline run.

The most recent run's results are also written to `outputs/_TestResults.md` at the
parent level for convenience (overwritten on each run).

**Template:** [`pipeline/templates/test-report-template.md`](../../pipeline/templates/test-report-template.md)

> This directory is gitignored in production. Contents are pipeline-generated.
