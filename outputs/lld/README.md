# outputs/lld/

This directory stores Low Level Design documents generated by the Spec Agent.

Each file is named `<work-item-id>_LLD.md` and corresponds to one pipeline run.

The most recent run's LLD is also written to `outputs/_LLD.md` at the parent level
for convenience (overwritten on each run).

**Template:** [`pipeline/templates/lld-template.md`](../../pipeline/templates/lld-template.md)

> This directory is gitignored in production. Contents are pipeline-generated.
